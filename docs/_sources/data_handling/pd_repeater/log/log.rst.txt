IoTデータ制御ツールログについて
===================================

IoTデータ制御ツールのログについては、「IoTデータ」→「ログ」タブから確認が行えます。

.. figure:: pd_log_pulldown.png
  :scale: 100%

閲覧したいログファイルをログ欄から選択します。

.. raw:: latex

   \clearpage

.. figure:: pd_log.png
  :scale: 100%

| ログファイルの選択を行うと、対象ログファイルの末尾が表示されます。
| ダウンロードボタンを押すことによりログファイルのダウンロードが行えます。
| また、「自動更新(30秒間隔)」をチェックすると30秒毎に更新されます。

.. raw:: latex

   \clearpage
