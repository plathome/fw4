はじめに¶

本書は、OpenBlocks シリーズを WEB ユーザーインターフェース(以下、WEB UI) で設定する方法を解説しています。本設定には、WEB ブラウザが使用可能なクライアント装置(PCやスマートフォン、タブレット等)が必要になります。
