ТјЦуХџуіХТЁІсЃјсЃ╝сЃЅсЂ«ТјЦуХџуіХТЁІсѓњУАеуц║сЂЌсЂЙсЂЎсђѓ

сЃљсЃ╝сѓИсЃДсЃ│тЈќтЙЌТЎѓтѕ╗сЃјсЃ╝сЃЅсЂ«сЃЉсЃЃсѓ▒сЃ╝сѓИсЂ«сЃљсЃ╝сѓИсЃДсЃ│сѓњтЈќтЙЌсЂЌсЂЪТюђухѓТЎѓтѕ╗сЂДсЂЎсђѓ
ТюђухѓТјЦуХџуб║УфЇТЎѓтѕ╗сЃјсЃ╝сЃЅсЂїТјЦуХџсЂЌсЂЪсЂ«сѓњуб║УфЇсЂЌсЂЪТюђухѓТЎѓтѕ╗сЂДсЂЎсђѓ
сЃЉсЃЃсѓ▒сЃ╝сѓИVer.сЃјсЃ╝сЃЅсЂ«сЃЉсЃЃсѓ▒сЃ╝сѓИ Ver.сЂ«СИђУдДсЂДсЂЎсђѓ
сѓ┐сѓцсЃасѓЙсЃ╝сЃ│сЃјсЃ╝сЃЅсЂ«сѓ┐сѓцсЃасѓЙсЃ╝сЃ│сЂДсЂЎсђѓ
сѓбсЃєсЃ│сѓисЃДсЃ│сЃјсЃ╝сЃЅсЂ«сѓбсЃєсЃ│сѓисЃДсЃ│сѓњУАеуц║сЂЌсЂЙсЂЎсђѓ
сЂфсЂісђЂсѓбсЃєсЃ│сѓисЃДсЃ│сЂїуЎ║ућЪсЂЌсЂдсЂёсѓІта┤тљѕсЂФсЂ»сђЂсѓбсЃєсЃ│сѓисЃДсЃ│УДБжЎцсЃюсѓ┐сЃ│сЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ
сЂЊсЂ«сЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЪта┤тљѕсђЂсЃјсЃ╝сЃЅсЂїсѓбсЃєсЃ│сѓисЃДсЃ│УДБжЎцТЕЪУЃйсЂФт»Йт┐юсЂЌсЂдсЂёсѓІсЂесЃјсЃ╝сЃЅсЂїС┐ЮТїЂсЂЎсѓІсѓбсЃєсЃ│сѓисЃДсЃ│ТЃЁта▒сѓњтЅіжЎцсЂЌсЂЙсЂЎсђѓ
MACсѓбсЃЅсЃгсѓ╣тљёсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сѓцсЃ│сѓ┐сЃЋсѓДсЃ╝сѓ╣сЂ«MACсѓбсЃЅсЃгсѓ╣сЂДсЂЎсђѓ
сЃАсЃбсЃфСй┐ућежЄЈ(MB)сЃАсЃбсЃфсЂ«Сй┐ућежЄЈсЂДсЂЎсђѓ
сЃЄсѓБсѓ╣сѓ»Сй┐ућежЄЈ(MB)сЃЄсѓБсѓ╣сѓ»сЂ«Сй┐ућежЄЈсЂДсЂЎсђѓ
IPсѓбсЃЅсЃгсѓ╣тљёсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сѓцсЃ│сѓ┐сЃЋсѓДсЃ╝сѓ╣сЂ«ТЃЁта▒сЂДсЂЎсђѓ

уиежЏєсЂФсЂцсЂёсЂдсЂ»сђЂConfigС╗ЦтцќсЂ«УеГт«џсѓњтЈѓуЁДсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

сѓхсЃЮсЃ╝сЃѕсЃГсѓ░#

т╝іуцЙсЂ«сѓхсЃЮсЃ╝сЃѕТІЁтйЊУђЁсЂІсѓЅсѓхсЃЮсЃ╝сЃѕсЃГсѓ░сѓњтЈќтЙЌсЂЎсѓІсѓѕсЂєУдЂУФІсЂЋсѓїсЂЪта┤тљѕсђЂсѓхсЃЮсЃ╝сЃѕсЃГсѓ░сЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓ

сЃјсЃ╝сЃЅсЂІсѓЅAirManage2сѓхсЃ╝сЃљсЂИсѓхсЃЮсЃ╝сЃѕсЃГсѓ░сѓњУ╗бжђЂсЂЎсѓІсЂЪсѓЂсЂФсђЂУ╗бжђЂсѓИсЃДсЃќС║ѕу┤ёсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓтИИТЎѓТјЦуХџсЂ«та┤тљѕсЂ»уЏ┤тЙїсЂФсђЂсѓфсЃ│сЃЄсЃъсЃ│сЃЅТјЦуХџсЂ«та┤тљѕсЂ»ТгАТјЦуХџТЎѓсЂФУ╗бжђЂсЂЌсЂЙсЂЎсђѓ

У╗бжђЂсЂїухѓС║єсЂЎсѓІсЂесђЂУ╗бжђЂт«їС║єТЎѓтѕ╗сђЂсѓхсѓцсѓ║сђЂсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЃюсѓ┐сЃ│сЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓсѓхсЃЮсЃ╝сЃѕсЃГсѓ░сѓњсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЂЎсѓІсЂФсЂ»сђЂсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓ

ухѓС║єсЂЎсѓІсЂФсЂ»сђЂсЃјсЃ╝сЃЅСИђУдДсЂИТѕ╗сѓІсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓ

Configу«Ауљє#

сЃјсЃ╝сЃЅсЂ«configсѓњу«АуљєсЂЎсѓІсЂФсЂ»сђЂConfigу«АуљєсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓсЃјсЃ╝сЃЅConfigСИђУдДућ╗жЮбсЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ

сЃјсЃ╝сЃЅсЂ«configсЂїСИђУдДсЂЋсѓїсЂЙсЂЎсђѓсѓфсЃфсѓИсЃісЃФСИќС╗БсЂ»сђЂConfigу«АуљєсЂДУеГт«џсЂЌсЂЪconfigсЂДсЂЎсђѓсЃјсЃ╝сЃЅсЂ«WEB-UIсЂДУеГт«џсѓњтцЅТЏ┤сЂЌсЂЪта┤тљѕсђЂТюђТќ░уЅѕсѓњтљФсѓЂсЂд3СИќС╗БсЂЙсЂДсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂЌсЂЙсЂЎсђѓУАеуц║сЂЎсѓІсЂФсЂ»сђЂУАеуц║сЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓсѓфсЃфсѓИсЃісЃФсЂесЂ«ти«тѕєсѓњУАеуц║сЂЎсѓІсЂФсЂ»сђЂсѓфсЃфсѓИсЃісЃФсЂесЂ«ти«тѕєсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓ1СИќС╗БтЅЇсЂесЂ«ти«тѕєсѓњУАеуц║сЂЎсѓІсЂФсЂ»сђЂ1СИќС╗БтЅЇсЂесЂ«ти«тѕєсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЙсЂЎсђѓ

сѓфсЃфсѓИсЃісЃФсЂ«configсѓњуиежЏєсЂЌС┐ЮтГўсЂЎсѓІсЂЊсЂесЂФсѓѕсЂБсЂдсђЂсЃфсѓ│сЃ│сЃЋсѓБсѓ░ућеУдйсЂФсЃЂсѓДсЃЃсѓ»сЂїтЁЦсѓісЂЙсЂЎсђѓ
