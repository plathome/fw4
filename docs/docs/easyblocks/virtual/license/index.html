ライセンス

仮想アプライアンスのEasyBlocksにて無償版及びライセンスファイルが無効な状態の場合、機能が制限(AirManageの使用権利やサポート、製品固有設定)されております。
有償版のライセンスファイルを仮想アプライアンスに適用することで対応した機能が有効化されます。

有償版ライセンスファイルの入手につきましては、EasyBlocks製品ページからの入手方法をご確認願います。

ライセンスファイルの適用処理#

  1. 対象の仮想アプライアンスのライセンスファイルを入手後、「メンテナンス」の「ライセンス」タブのアップロード項目にてライセンスファイルをアップロード(ファイル選択後、実行ボタン)します。
  2. アップロード後、アップロードしたライセンスファイルの情報が表示されますので問題がなければ「操作」項目にて適用ボタンを押します。
    内容に関して問題がある場合には、削除ボタンを押し入手元へご連絡ください。
  3. 適用処理が完了すると適用しているライセンスファイルの情報が表示されます。
