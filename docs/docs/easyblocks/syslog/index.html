EasyBlocks Syslog

SSDを搭載した最新のSyslogアプライアンス製品(一部HDDモデルあり)です。高いログ受信性能をもち、多様な機器からのログ収集及び搭載SSDに応じた長期保存が可能です。

EasyBlocks初期設定共通#

初期設定は必ず以下の順番で設定してください。
項目5.のネットワーク設定ではシステム再起動必須になります。
また、項目6.はEasyBlocks本体が項目5.で設定されたIPに変更になるので、操作前に本番ネットワークへの接続と操作用WEBブラウザのURL変更を忘れないでください。

  1. EasyBlocks KVM設定(EasyBlocks ProLineモデルのみ)
  2. WEB-UI接続準備
  3. ソフトウエア使用許諾
  4. 管理者アカウント登録
  5. 時刻同期
  6. ネットワークの設定
  7. 遠隔管理AirManage
caution
  • Syslogサーバーの場合、ストレージ容量が不十分な場合動作不安定になるので、ここでストレージ監視を有効にする事をお奨めします。
