ホスト追加(Syslog&NetworkReporter用)

ローカルネットワーク等のDNSを用いずホスト名とIPアドレスの解決を行うhostsファイル内容について追記が行えます。

設定項目は以下のとおりです。

項目説明
ホスト追加IPアドレスとホスト名を1行ずつ記載します。
IPアドレスとホスト名の区切りはスペースにてを用いてください。
なお、ホスト名は複数記載することが可能です
【書式】
＜IPアドレス＞ ＜ホスト名＞
Ex.)
192.168.100.1 gateway
192.168.100.1 gateway.example.org
