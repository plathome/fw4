тЅ▓тйЊсЃЮсЃфсѓисЃ╝т»ЙУ▒А/т»ЙУ▒Атцќт┐Ёжаѕсђїт»ЙУ▒АсђЇсЂФсЂЎсѓІсЂесђЂтЅ▓тйЊсЃЮсЃфсѓисЃ╝сЂ«т»ЙУ▒АсЂФсЂфсѓісЂЙсЂЎсђѓ
сђїт»ЙУ▒АтцќсђЇсЂФсЂЎсѓІсЂесђЂтЅ▓тйЊсЃЮсЃфсѓисЃ╝сЂ«т»ЙУ▒АтцќсЂФсЂфсѓісЂЙсЂЎсђѓ
тЏ║т«џ IP сѓбсЃЅсЃгсѓ╣10жђ▓ТЋ░сЃЅсЃЃсЃѕтї║тѕЄсѓіС╗╗ТёЈтЏ║т«џ IP сѓбсЃЅсЃгсѓ╣УеГт«џсѓњУАїсЂёсЂЙсЂЎсђѓ

С╗ЦСИІсђЂуиежЏєТќ╣Т│ЋсЂФсЂцсЂёсЂдУДБУфгсЂЌсЂЙсЂЎсђѓ

уиежЏє#

сЃЕсѓИсѓфсЃюсѓ┐сЃ│сЂ«уиежЏєсѓњжЂИТіъсЂЎсѓІсЂесђЂуиежЏєтЈ»УЃйсЂфу«ЄТЅђсЂ«тЉесѓісЂїуиЉУЅ▓сЂФсЂфсѓісЂЙсЂЎсђѓ
уиЉУЅ▓сЂФсЂфсЂБсЂЪу«ЄТЅђсЂФсѓФсЃ╝сѓйсЃФсѓњтљѕсѓЈсЂЏсЂдсђЂуиежЏєсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

У┐йтіа#

Тќ░УдЈсЂФсЃЏсѓ╣сЃѕсѓњУ┐йтіасЂЎсѓІта┤тљѕсЂ»сђЂУ┐йтіасЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
У┐йтіасЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂесђЂтЁЦтіЏТгёсЂїУ┐йтіасЂЋсѓїсЂЙсЂЎсђѓ

УцЄУБй#

ТЌбтГўсЂ«УеГт«џсѓњУцЄУБйсЂЌсЂЪсЂёта┤тљѕсЂ»сђЂсЃєсЃ╝сЃќсЃФсЂ«тЈ│уФ»сЂФсЂѓсѓІУцЄУБйсѓњжЂИТіъсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
жЂИТіъсЂЌсЂЪУАїсЂ«сѓ│сЃћсЃ╝сЂїСйюТѕљсЂЋсѓїсЂЙсЂЎсђѓ

тЅіжЎц#

сЃЏсѓ╣сЃѕТЃЁта▒сѓњтЅіжЎцсЂЎсѓІта┤тљѕсЂ»сђЂсЃєсЃ╝сЃќсЃФсЂ«тЈ│уФ»сЂФсЂѓсѓІтЅіжЎцсѓњсѓ»сЃфсЃЃсѓ»сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
сЃфсЃ╝сѓ╣ТЃЁта▒сЂїсЂфсЂёта┤тљѕсЂ»сђЂсЂЮсЂ«УАїсЂ«сЃЏсѓ╣сЃѕТЃЁта▒сЂ»тЅіжЎцсЂЋсѓїсЂЙсЂЎсђѓ
сЃфсЃ╝сѓ╣ТЃЁта▒сЂїсЂѓсѓІта┤тљѕсЂ»сђЂсЂЮсЂ«УАїсЂ«сЃфсЃ╝сѓ╣ТЃЁта▒С╗ЦтцќсЂ«ТЃЁта▒сЂїтЅіжЎцсЂЋсѓїсЂЙсЂЎсђѓ

тЏ║т«џIPсЂ«ТЅЋсЂётЄ║сЂЌУеГт«џ#

С╗╗ТёЈсЂ« IP сѓбсЃЅсЃгсѓ╣сѓњтЏ║т«џсЂДТЅЋсЂётЄ║сЂЌсЂЪсЂёта┤тљѕсЂ»сђЂсЃЏсѓ╣сЃѕТЃЁта▒сЂ«уиежЏєсЂДтЏ║т«џ IP сѓбсЃЅсЃгсѓ╣ТгёсЂФсђЂТЅЋсЂётЄ║сЂЌсЂЪсЂё IP сѓбсЃЅсЃгсѓ╣сѓњтЁЦтіЏсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
сЂЊсЂ«жаЁуЏ«сЂїуЕ║ТгёсЂ«та┤тљѕсЂ»сђЂтЏ║т«џ IP сѓбсЃЅсЃгсѓ╣УеГт«џсЂ»УАїсѓЈсѓїсЂЙсЂЏсѓЊсђѓ

тЏ║т«џIPсЂ«ТЅЋсЂётЄ║сЂЌсЂетЅ▓тйЊсЃЮсЃфсѓисЃ╝т»ЙУ▒АУеГт«џсЂ«уФХтљѕ#

тЏ║т«џ IP сѓбсЃЅсЃгсѓ╣ТЅЋсЂётЄ║сЂЌУеГт«џсЂетЅ▓тйЊсЃЮсЃфсѓисЃ╝т»ЙУ▒АУеГт«џсѓњтљїТЎѓсЂФУАїсЂБсЂЪта┤тљѕсђЂтЏ║т«џ IPсѓбсЃЅсЃгсѓ╣ТЅЋсЂётЄ║сЂЌУеГт«џсЂїтёфтЁѕсЂЋсѓїсЂЙсЂЎсђѓ
тЅ▓тйЊсЃЮсЃфсѓисЃ╝т»ЙУ▒АУеГт«џсЂїт»ЙУ▒АсЂФсЂфсЂБсЂдсЂёсЂдсѓѓсђЂт»ЙУ▒АтцќсЂФсЂфсЂБсЂдсЂёсЂдсѓѓтЏ║т«џ IP сѓбсЃЅсЃгсѓ╣сѓњТЅЋсЂётЄ║сЂЌсЂЙсЂЎсђѓ

С┐ЮтГў#

сЃЏсѓ╣сЃѕТЃЁта▒сЂ«уиежЏєсѓњухѓсЂѕсЂЪсѓЅсђЂС┐ЮтГўсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
С┐ЮтГўсЃюсѓ┐сЃ│сѓњТі╝сЂЋсЂфсЂёсЂеуиежЏєсЂЌсЂЪтєЁт«╣сЂ»С┐ЮтГўсЂЋсѓїсЂЙсЂЏсѓЊсђѓ
С┐ЮтГўсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЪтЙїсЂ»сђЂсѓхсЃ╝сЃЊсѓ╣сЂ«сЃфсЃГсЃ╝сЃЅсѓњт«ЪУАїсЂЌсЂдсђЂУеГт«џсѓњтЈЇТўасЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
Рђ╗уиежЏєсЃбсЃ╝сЃЅсЂІсѓЅжќ▓УдДсЃбсЃ╝сЃЅсЂФтцЅТЏ┤сђЂжќ▓УдДсЃбсЃ╝сЃЅсЂІсѓЅуиежЏєсЃбсЃ╝сЃЅсЂФтцЅТЏ┤сЂЌсЂдсѓѓсђЂуиежЏєСИГсЂ«тєЁт«╣сЂ»С┐ЮТїЂсЂЋсѓїсЂЙсЂЎсђѓсЂЪсЂасЂЌсђЂТюгУБЁуй«сЂФС┐ЮтГўсЂЋсѓїсЂдсЂёсѓІсѓЈсЂЉсЂДсЂ»сЂфсЂёсЂ«сЂДсђЂУеГт«џсЂ«С┐ЮтГўсѓњУАїсЂєта┤тљѕсЂ»С┐ЮтГўсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
