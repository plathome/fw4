сѓесѓ»сѓ╣сЃЮсЃ╝сЃѕ#

DHCP УеГт«џсЃЋсѓАсѓцсЃФсЂИсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂЎсѓІТЕЪУЃйсЂДсЂЎсђѓ
сѓесѓ»сѓ╣сЃЮсЃ╝сЃѕсЂЌсЂЪсЃЋсѓАсѓцсЃФсѓњсѓцсЃ│сЃЮсЃ╝сЃѕсЂЎсѓІсЂЊсЂесЂФсѓѕсЂБсЂдсђЂжЂјтј╗сЂ«УеГт«џсЂФТѕ╗сЂЎсЂЊсЂесЂїтЈ»УЃйсЂДсЂЎсђѓ
сѓесѓ»сѓ╣сЃЮсЃ╝сЃѕсЂЋсѓїсѓІсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЃЋсѓАсѓцсЃФсЂ»сђЂТЊЇСйюуФ»ТюФсЂ«сЃђсѓдсЃ│сЃГсЃ╝сЃЅсЃЄсѓБсЃгсѓ»сЃѕсЃфсЂФС┐ЮтГўсЂЋсѓїсЂЙсЂЎсђѓ
Рђ╗УеГт«џсѓњТЏ┤Тќ░сЂЌсЂЪТЎѓсЂ»т┐ЁсЂџсѓесѓ»сѓ╣сЃЮсЃ╝сЃѕсЂЌсЂдсЃљсЃЃсѓ»сѓбсЃЃсЃЌсѓњтЈќсЂБсЂдсЂЈсЂасЂЋсЂёсђѓ

сѓцсЃ│сЃЮсЃ╝сЃѕ#

DHCP УеГт«џсѓњсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЃЋсѓАсѓцсЃФсЂІсѓЅсЃфсѓ╣сЃѕсѓбсЂЎсѓІТЕЪУЃйсЂДсЂЎсђѓ
сѓцсЃ│сЃЮсЃ╝сЃѕсЂЎсѓІсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЃЋсѓАсѓцсЃФсЂ»сђЂТЊЇСйюуФ»ТюФсЂ«С╗╗ТёЈсЃЄсѓБсЃгсѓ»сЃѕсЃфсЂФС┐ЮтГўсЂЋсѓїсЂЪсЃЋсѓАсѓцсЃФсѓњСй┐сЂёсЂЙсЂЎсђѓ
сѓесѓ»сѓ╣сЃЮсЃ╝сЃѕсЂЌсЂЪсЃЋсѓАсѓцсЃФсѓњсѓцсЃ│сЃЮсЃ╝сЃѕсЂЎсѓІсЂЊсЂесЂФсѓѕсЂБсЂдсђЂжЂјтј╗сЂ«УеГт«џсЂФТѕ╗сЂЎсЂЊсЂесЂїтЈ»УЃйсЂДсЂЎсђѓ

сЃЏсѓ╣сЃѕу«АуљєТЃЁта▒сЂ«сЃљсЃЃсѓ»сѓбсЃЃсЃЌсЃ╗сЃфсѓ╣сЃѕсѓб#

сѓесѓ»сѓ╣сЃЮсЃ╝сЃѕ#

сЃЏсѓ╣сЃѕТЃЁта▒сѓњ CSV сЃЋсѓАсѓцсЃФсЂФсѓесѓ»сѓ╣сЃЮсЃ╝сЃѕ№╝ѕсЃљсЃЃсѓ»сѓбсЃЃсЃЌ№╝ЅсЂЎсѓІсЂЊсЂесЂїтЄ║ТЮЦсЂЙсЂЎсђѓ
