OpenBlocks IX9

OpenBlocksシリーズでエッジコンピューティング向け高速タイプです。
小型堅牢省エネ型のオフィスサーバー。

IX9パッケージ内容#

OpenBlocks IX9の標準品構成は以下の通りです。

IX9本体 1台#

USB Type-A - Micro USBケーブル 1本#
