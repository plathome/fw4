Device AddressBLEсЃЄсЃљсѓцсѓ╣сЂ«ТцютЄ║сЂ«жџЏсЂФтЈќтЙЌсЂЌсЂЪсѓбсЃЅсЃљсѓ┐сѓцсѓ║сЃЄсЃ╝сѓ┐сѓњсѓѓсЂесЂФсЃЄсЃљсѓцсѓ╣сѓбсЃЅсЃгсѓ╣сѓњУАеуц║сЂЌсЂЙсЂЎсђѓ
LuaсЃЌсЃГсЃЋсѓАсѓцсЃФBLEсЃЄсЃљсѓцсѓ╣сЂ«сѓ╗сЃ│сѓхсЃ╝сЂ«LuaсЂ«сЃЌсЃГсЃЋсѓАсѓцсЃФтљЇсѓњТїЄт«џсЂЌсЂЙсЂЎсђѓ
сЂЪсЂасЂ«сЃЊсЃ╝сѓ│сЃ│сЂесЂЌсЂдТЅ▒сЂєта┤тљѕсЂФсЂ»тЁЦтіЏсЂ»СИЇУдЂсЂДсЂЎсђѓ
УеГт«џтЈ»УЃйсЂфLuaсЃЌсЃГсЃЋсѓАсѓцсЃФсЂ»т»Йт┐юсѓ╗сЃ│сѓхсЃ╝сЃфсѓ╣сЃѕсЂІсѓЅсЂћуб║УфЇСИІсЂЋсЂёсђѓ
сЃдсЃ╝сѓХсЃ╝сЃАсЃбBLEсЃЄсЃљсѓцсѓ╣сЂ«ТцютЄ║сЂ«жџЏсЂФтЈќтЙЌсЂЌсЂЪсѓбсЃЅсЃљсѓ┐сѓцсѓ║сЃЄсЃ╝сѓ┐сѓњсѓѓсЂесЂФсЃЄсЃљсѓцсѓ╣тљЇсѓњсЃЄсЃЋсѓЕсЃФсЃѕсЂДУеГт«џсЂЌсЂЙсЂЎсђѓ
сЂЊсЂ«сЃЋсѓБсЃ╝сЃФсЃЅсЂ»уиежЏєтЈ»УЃйсЂДсЂЎсЂ«сЂДсѓ╗сЃ│сѓхсЃ╝сЂ«тѕцтѕЦтљЇсЂфсЂЕУЄфућ▒сЂФСй┐сЂБсЂдСИІсЂЋсЂёсђѓ
сЃџсѓбсЃфсЃ│сѓ░сЃџсѓбсЃфсЃ│сѓ░сѓњт┐ЁУдЂсЂесЂЎсѓІсЃЄсЃљсѓцсѓ╣сЂ«сЂ┐сЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂдСИІсЂЋсЂёсђѓ(т»Йт┐юсѓ╗сЃ│сѓхсЃ╝сЃфсѓ╣сЃѕсЂ«GATTсЂїсђЄсЂ«сЃЄсЃљсѓцсѓ╣)

сЂфсЂісђЂуЎ╗жї▓ТИѕсЂ┐сЂ«BLEсЃЄсЃљсѓцсѓ╣сЂФсЂцсЂёсЂдсЂ»сђЂТюђСИІТ«хсЂ«СИђУдДсЂФТюђтѕЮсЂІсѓЅУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ
сЂЙсЂЪсђЂСИђУдДсЂІсѓЅсЃЄсЃљсѓцсѓ╣сЂ«тЅіжЎцтЈісЂ│тљёТЃЁта▒сЂ«ТЏ┤Тќ░сЂїУАїсЂѕсЂЙсЂЎсђѓ

EnOceanуЎ╗жї▓#

OpenBlocksсЂФсѓфсЃЌсѓисЃДсЃ│сЂ«EnOceanТІАт╝хсЃбсѓИсЃЦсЃ╝сЃФсѓњТљГУ╝ЅсЂЌсђЂEnOceanсЂ«сЃЄсЃљсѓцсѓ╣ТЃЁта▒сѓњтЈќтЙЌсЂЎсѓІсЂЪсѓЂсЂ«уЎ╗жї▓сЂДсЂЎсђѓ
EnOceanсЂ»BLEсЂ«сЃЊсЃ╝сѓ│сЃ│сЂетљїТДўсЂ«тІЋСйюсЂфсЂ«сЂДсђЂсЂЊсЂЊсЂДсЂ»тЈЌС┐АсЂЎсѓІEnOceanсЃЄсЃљсѓцсѓ╣сЂ«уЎ╗жї▓сЂесЂфсѓісЂЙсЂЎсђѓ

EnOceanуЎ╗жї▓сѓ┐сЃќсЂФсЂѓсѓІт┐ЁУдЂжаЁуЏ«сѓњтЁЦтіЏсЂЌсЂдСИІсЂЋсЂёсђѓ
IDсѓёEEPсЂ«ТЃЁта▒сЂ»УБйтЊЂсЃЕсЃЎсЃФсЂфсЂЕсЂФУеўУ╝ЅсЂЋсѓїсЂдсЂёсѓІсЂ«сЂДтЈѓуЁДСИІсЂЋсЂёсђѓ

жаЁуЏ«УфгТўј
сЃЄсЃљсѓцсѓ╣IDуЎ╗жї▓т»ЙУ▒АсЂ«EnOceanсЃЄсЃљсѓцсѓ╣сЂ«сЃЄсЃљсѓцсѓ╣IDсѓњУеГт«џсЂЌсЂЙсЂЎсђѓ
сЃдсЃ╝сѓХсЃ╝сЃАсЃбсЂЊсЂ«сЃЋсѓБсЃ╝сЃФсЃЅсЂ»уиежЏєтЈ»УЃйсЂДсЂЎсЂ«сЂДсѓ╗сЃ│сѓхсЃ╝сЂ«тѕцтѕЦтљЇсЂфсЂЕУЄфућ▒сЂФСй┐сЂБсЂдСИІсЂЋсЂёсђѓ
EEP(ТЕЪтЎеТЃЁта▒сЃЌсЃГсЃЋсѓАсѓцсЃФ)уЎ╗жї▓т»ЙУ▒АсЂ«EnOceanсЃЄсЃљсѓцсѓ╣сЂ«EEP(сЃЌсЃГсЃЋсѓАсѓцсЃФ)сѓњУеГт«џсЂЎсѓІсЂЊсЂесЂїсЂДсЂЇсЂЙсЂЎсђѓ
сЂЊсЂ«EEPсЂФТГБсЂЌсЂёсЃЌсЃГсЃЋсѓАсѓцсЃФсѓ│сЃ╝сЃЅсѓњтЁЦтіЏсЂЎсѓІсЂесђЂEnOceanсЃЄсЃљсѓцсѓ╣сЂ«уе«жАъсЂФсѓѕсЂБсЂдjsonТќЄтГЌтѕЌсЂФжЂЕтѕЄсЂфТИЕт║дсЃ╗Т╣┐т║дуГЅсЂ«сѓГсЃ╝тљЇсЂїУеГт«џсЂЋсѓїсЂЙсЂЎсђѓ
GP(Generic Profile)сЂ«та┤тљѕсЂ»LUAсЂ«GPсЃЌсЃГсЃЋсѓАсѓцсЃФтљЇсѓњуЎ╗жї▓сЂЌсЂЙсЂЎсђѓУеГт«џтЈ»УЃйсЂфGPсЃЌсЃГсЃЋсѓАсѓцсЃФтљЇсЂ»GPсѓ╗сЃ│сѓхсЃ╝сЂІсѓЅсЂћуб║УфЇСИІсЂЋсЂёсђѓ

сЂфсЂісђЂуЎ╗жї▓ТИѕсЂ┐сЂ«EnOceanсЃЄсЃљсѓцсѓ╣сЂФсЂцсЂёсЂдсЂ»сђЂТюђСИІТ«хсЂ«СИђУдДсЂФТюђтѕЮсЂІсѓЅУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ
сЂЙсЂЪсђЂСИђУдДсЂІсѓЅсЂ»сЃЄсЃљсѓцсѓ╣сЂ«тЅіжЎцтЈісЂ│тљёТЃЁта▒сЂ«ТЏ┤Тќ░сЂїУАїсЂѕсЂЙсЂЎсђѓ

сѓцсЃ│сЃЮсЃ╝сЃѕ/сѓесѓ»сѓ╣сЃЮсЃ╝сЃѕ

сЃЊсЃ╝сѓ│сЃ│тъІуцЙтЊАУе╝сЂфсЂЕТ▓бт▒▒сЂ«EnOceanсЃЄсЃљсѓцсѓ╣сѓњуЎ╗жї▓сЂЎсѓІта┤тљѕсђЂУцЄТЋ░сЂ«OpenBlocksсЂДтљїсЂўСйюТЦГсѓњу╣░сѓіУ┐ћсЂЎсЂ«сЂ»тцДтцЅсЂфСйюТЦГсЂФсЂфсѓісЂЙсЂЎсђѓ
сЂЊсЂ«ТДўсЂфта┤тљѕсЂФсЃЊсЃ╝сѓ│сЃ│ТЃЁта▒сѓњуЎ╗жї▓сЂЌсЂЪOpenBlocksСИісЂДуЎ╗жї▓ТЃЁта▒сѓњсѓесѓ»сѓ╣сЃЮсЃ╝сЃѕсЂЌсђЂС╗ќсЂ«OpenBlocksсЂДсѓцсЃ│сЃЮсЃ╝сЃѕсЂЌсЂЙсЂЎсђѓ

жаЁуЏ«УфгТўј
сѓесѓ»сѓ╣сЃЮсЃ╝сЃѕOpenBlocksсЂФС┐ЮтГўсЂЌсЂЪEnOceanсЃЄсЃљсѓцсѓ╣ТЃЁта▒сѓњJSONсЃЋсѓАсѓцсЃФсЂесЂЌсЂдсѓесѓ»сѓ╣сЃЮсЃ╝сЃѕсЂЌсЂЙсЂЎсђѓ
сѓцсЃ│сЃЮсЃ╝сЃѕJSONсЃЋсѓАсѓцсЃФсѓњтЁЦтіЏсЂЌсђЂEnOceanсЃЄсЃљсѓцсѓ╣ТЃЁта▒сѓњсѓцсЃ│сЃЮсЃ╝сЃѕсЂЌсЂЙсЂЎсђѓ
С┐ЮтГўсѓцсЃ│сЃЮсЃ╝сЃѕсЂЌсЂЪEnOceanсЃЄсЃљсѓцсѓ╣сЂ«JSONсЃЋсѓАсѓцсЃФсЂ«тєЁт«╣сѓњТюгуГљСйЊсЂИС┐ЮтГўсЂЌсЂЙсЂЎсђѓ

JSONсЃЋсѓАсѓцсЃФсЂ«тєЁт«╣сЂ»WEB UIсЂ«сЃљсЃ╝сѓИсЃДсЃ│сЂФсѓѕсЂБсЂдуЋ░сЂфсѓІта┤тљѕсЂїсЂѓсѓісЂЙсЂЎсђѓ

Modbus2сЃЈсЃ│сЃЅсЃЕ#
