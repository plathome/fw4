OpenBlocksсѓисЃфсЃ╝сѓ║сЂ«ТеЎТ║ќсЃЄсЃ╝сѓ┐тЈјжЏєсѓбсЃЌсЃфсѓ▒сЃ╝сѓисЃДсЃ│(PD Handler)сѓњућесЂёсЂџсђЂсЃдсЃ╝сѓХсЃ╝сЂїуІгУЄфсЂФућеТёЈсЂЎсѓІсѓбсЃЌсЃфсѓ▒сЃ╝сѓисЃДсЃ│сЂФсѓѕсѓісЃЄсЃ╝сѓ┐сѓњсЃЈсЃ│сЃЅсЃфсЃ│сѓ░сЂЌсђЂPD RepeaterсЂежђБТљ║сЂЋсЂЏсѓІсЂЊсЂесЂїсЂДсЂЇсЂЙсЂЎсђѓ
IoTсЃЄсЃ╝сѓ┐РЄњжђЂтЈЌС┐АУеГт«џсѓ┐сЃќсЂ«сЃЄсЃљсѓцсѓ╣УеГт«џ№╝ѕсЃдсЃ╝сѓХсЃ╝т«џуЙЕ№╝ЅсЃџсѓцсЃ│сѓѕсѓісђЂсЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂ«УеГт«џсѓњУАїсЂєсЂЊсЂесЂїтЄ║ТЮЦсЂЙсЂЎсђѓ
сЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сѓњтѕЕућесЂЎсѓІсЂЪсѓЂсЂФсЂ»сђЂсѓхсЃ╝сЃЊсѓ╣РЄњDevуЎ╗жї▓РЄњсЃдсЃ╝сѓХсЃ╝сЃЄсЃљсѓцсѓ╣уЎ╗жї▓сѓ┐сЃќсѓѕсѓісђЂсѓхсЃ╝сЃЊсѓ╣ТЕЪУЃйсЂ«сЃЄсЃљсѓцсѓ╣уЎ╗жї▓сЂесЂЌсЂдсЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂїуЎ╗жї▓сЂЋсѓїсЂдсЂёсѓІт┐ЁУдЂсЂїсЂѓсѓісЂЙсЂЎсђѓУЕ│у┤░сЂ»сђЂсЃЄсЃљсѓцсѓ╣уЎ╗жї▓ТЕЪУЃйсЂ«UserсЃЄсЃљсѓцсѓ╣уЎ╗жї▓сѓњтЈѓуЁДсЂЌсЂдСИІсЂЋсЂёсђѓ
сЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂ«уЎ╗жї▓сЂесЂ»сђЂсЃЄсЃљсѓцсѓ╣уЋфтЈисѓњтЅ▓сѓіТї»сѓісЃАсЃбТЃЁта▒сѓњС╗ўСИјсЂЎсѓІСйюТЦГсЂДсЂЎсђѓ

уЎ╗жї▓ТИѕсЂ«сЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂїтГўтюесЂЌсЂдсЂёсѓІта┤тљѕсђЂтѕЮТюЪуіХТЁІсЂДсЂ»СИітЏ│сЂ«сѓѕсЂєсЂФсЂфсЂБсЂдсЂёсЂЙсЂЎсђѓ(сЃЄсЃљсѓцсѓ╣сЂї1тђІуЎ╗жї▓сЂЋсѓїсЂдсЂёсѓІта┤тљѕсЂДсЂЎсђѓ)

УеГт«џжаЁуЏ«УфгТўј
сЃЄсЃљсѓцсѓ╣уЋфтЈиOpenBlocksсѓисЃфсЃ╝сѓ║сЂ«WEB-UIтєЁсЂДу«АуљєсЂЌсЂдсЂёсѓІуЋфтЈисЂДсЂЎсђѓтцЅТЏ┤сЂ»сЂДсЂЇсЂЙсЂЏсѓЊсђѓ
сЃЄсЃ╝сѓ┐жќЊт╝ЋжќЊжџћсЃЄсЃ╝сѓ┐сѓњжќЊт╝ЋсЂЈсЂЪсѓЂсЂ«тЁЦтіЏсЃЄсЃ╝сѓ┐сѓњтЈЌсЂЉтЈќсѓЅсЂфсЂёТЎѓжќЊсѓњmsecтЇўСйЇсЂДУеГт«џсЂЌсЂЙсЂЎсђѓ
0сЂ«та┤тљѕсђЂжќЊт╝ЋсЂЇсЂ»УАїсѓЈсѓїсЂЙсЂЏсѓЊсђѓ
сЃљсЃЃсЃЋсѓАсЃ╝сѓхсѓцсѓ║сЃЄсЃ╝сѓ┐сЂ«ТюђтцДсѓхсѓцсѓ║сѓњУеГт«џсЂЌсЂЙсЂЎсђѓтЇўСйЇсЂ»сЃљсѓцсЃѕсЂДсЂЎсђѓ
тЈЌС┐АУеГт«џPD RepeaterсѓњС╗ІсЂЌсЂдсѓ»сЃЕсѓдсЃЅсЂІсѓЅсЂ«тѕХтЙАсЃАсЃЃсѓ╗сЃ╝сѓИсѓњтЈЌсЂЉсѓІСИІТхЂТќ╣тљЉтѕХтЙАсѓњУАїсЂєсЂІтљдсЂІсѓњжЂИТіъсЂЌсЂЙсЂЎсђѓ
УЄфСйюсЂ«HandlerуГЅсѓњућесЂёсЂдсЂ«СИІТхЂТќ╣тљЉтѕХтЙАсѓњУАїсЂєта┤тљѕсЂФсЂцсЂёсЂдсЂ»сђЂсЂЊсЂАсѓЅсѓњсЂћуб║УфЇсЂЈсЂасЂЋсЂёсђѓ
сЃдсЃ╝сѓХсЃ╝сЃАсЃбуЎ╗жї▓сЂЋсѓїсЂЪсЃЄсЃљсѓцсѓ╣сЂФсЂдУеГт«џсЂЋсѓїсЂЪсЃАсЃбТЃЁта▒сѓњУАеуц║сЂЌсЂЙсЂЎсђѓ
жђЂС┐АтЁѕУеГт«џСй┐ућесЂЎсѓІсѓњжЂИТіъсЂЌсЂЪжђЂС┐АтЁѕсЂФт»ЙсЂЌсЂдсЃЂсѓДсЃЃсѓ»сЃюсЃЃсѓ»сѓ╣сЂїжЂИТіъсЂДсЂЇсѓІсѓѕсЂєсЂФсЂфсѓісЂЙсЂЎсђѓ
