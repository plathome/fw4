info

ログに出力されるJSONデータは順不同です。

RAWデータ#

■ データサンプル :

{
"8D": "0123456789AB",
"time": "2018-11-21T12:34:56+09:00",
"memo": "raw data",
"datatype": "raw_data",
"raw_data":[
{
"EPC": "80",
"PDC": 1,
"EDT": "30"
},
{
"EPC": "88",
"PDC": 1,
"EDT": "41"
}
]
}
RAWデータ
#JSONキー内容データ型常駐補足
18D製造番号String〇
2timeデータ取得日時String〇ISO8601拡張書式
3datatypeデータータイプString〇'raw_data'
4raw_dataRAW データJSONオブジェクト配列〇
5EPCECHONETプロパティString〇
6PDCプロパティデータカウンタInteger〇
7EDTECHONETプロパティ値データString〇
exユーザー設定ユーザー設定内容-固定情報付与(JSON)の設定内容がデータへ追加されます。
