ビーコン#

ビーコン送信設定時#

■ データサンプル　:

{
"time": "2017-12-08T15:00:04.549+09:00",
"deviceId": "e9c8dd35ee18",
"appendixInfo": "G8H00012",
"rssi": -88,
"type": "iBeacon",
"data": "0201040c0946434c20426561636f6e31",
"localname": "beacon",
"status": "in"
}
#JSONキー内容データ型常駐補足
1timeデータ取得日時String〇ISO8601拡張書式
2deviceIdデバイスIDString〇デバイスアドレスから ':' を除き、小文字化した値
3rssi受信信号強度Integer〇
4typeビーコン種別StringWEB-UIにて表示設定
5dataペイロードデータString16進ダンプデータ
6localnameローカル名StringWEB-UIにて表示設定
7statusビーコンステータスStringビーコン制御タイプにより表示。('in' または 'out')
exユーザー設定ユーザー設定内容-BLEビーコン送信設定の固定情報付与(JSON)の設定内容(ここでは「{"appendixInfo": "G8H00012"}」)がデータへ追加されます。

リコー CO2センサ (Luaプロファイル:RICOH_CO2)#

info

センサの測定間隔等は、RICOH社提供のAndroid™端末用ソフトウェアRICOH EH Environment Sensor for Android™で設定変更してください。

info

センサは、環境系データを送信した5秒後にCO2データを送信します。
センサをONにした直後の1回目の通信においては、環境系データのみ送信されます。 センサをONにしてから2回目の環境系データが送信された後、初回のCO2データが送信されます。

BLEデバイス設定の取得時間間隔[sec]を5秒未満にしてください。
5秒以上に設定した場合、環境系データとCO2データのどちらか一方のデータしか取得できません。
