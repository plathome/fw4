СИђт║дсЂДсѓѓУхитІЋсЂЌсЂЪсЂЊсЂесЂ«сЂѓсѓІсѓ│сЃ│сЃєсЃісЂ«уіХТ│ЂсЂ«уб║УфЇсђЂУхитІЋсЃ╗тЂюТГбуГЅсЂ«тѕХтЙАсѓњDockerРЄњсѓ│сЃ│сЃєсЃіСИђУдДсѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ

Рќасѓ│сЃ│сЃєсЃіуе╝тЃЇСИГсЂ«та┤тљѕ

сѓ│сЃ│сЃєсЃісЂїуе╝тЃЇСИГсЂ«та┤тљѕсђЂтЂюТГбсЂЎсѓІсЂЊсЂесЂїсЂДсЂЇсЂЙсЂЎсђѓ
тЂюТГбсЂЌсЂЪсЂёсѓ│сЃ│сЃєсЃісЂ«тЂюТГбсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂФсѓѕсѓісђЂтЂюТГбсЂЎсѓІсЂЊсЂесЂїтЈ»УЃйсЂДсЂЎсђѓ

сЂфсЂісђЂуе╝тЃЇСИГсЂ«сѓ│сЃ│сЃєсЃісЂ»тЁгжќІсЃЮсЃ╝сЃѕтЈісЂ│IPсѓбсЃЅсЃгсѓ╣сЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ

Рќасѓ│сЃ│сЃєсЃітЂюТГбСИГсЂ«та┤тљѕ

сѓ│сЃ│сЃєсЃітЂюТГбСИГсЂ«та┤тљѕсђЂУхитІЋтЈісЂ│тЅіжЎцсЂїУАїсЂѕсЂЙсЂЎсђѓ
УхитІЋсЂЋсЂЏсѓІта┤тљѕсЂФсЂ»сђЂУхитІЋсЂЌсЂЪсЂёсѓ│сЃ│сЃєсЃісЂ«УхитІЋсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂДУхитІЋсЂЌсЂЙсЂЎсђѓ

т»ЙУ▒Асѓ│сЃ│сЃєсЃісѓњтЅіжЎцсЂЎсѓІта┤тљѕсЂФсЂ»сђЂтЅіжЎцсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

Dockerсѓ│сЃ│сЃєсЃісЃфсѓйсЃ╝сѓ╣УАеуц║#

СИђт║дсЂДсѓѓУхитІЋсЂЌсЂЪсЂЊсЂесЂ«сЂѓсѓІсѓ│сЃ│сЃєсЃісЂ«сЃфсѓйсЃ╝сѓ╣уіХТ│Ђуб║УфЇсѓњDockerРЄњсЃфсѓйсЃ╝сѓ╣сѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ
тЂюТГбСИГсЂ«сѓ│сЃ│сЃєсЃісЂ»СИђУдДсЂФУАеуц║сЂЋсѓїсЂЙсЂЎсЂїсђЂтЂюТГбсЂЌсЂдсЂёсѓІсЂЪсѓЂтљёсЃфсѓйсЃ╝сѓ╣сЂ»ТюфСй┐ућеуіХТЁІсЂесЂфсѓісЂЙсЂЎсђѓ

сЂЙсЂЪсђЂТюгжаЁсЂФсЂдсЃфсѓйсЃ╝сѓ╣сѓњуб║УфЇсЂЌсђЂтљёсѓ│сЃ│сЃєсЃісЂ«сЃфсѓйсЃ╝сѓ╣сЃЂсЃЦсЃ╝сЃІсЃ│сѓ░сѓњУАїсЂБсЂдСИІсЂЋсЂёсђѓ

Dockerсѓ│сЃ│сЃєсЃісЃфсѓйсЃ╝сѓ╣УеГт«џТЏ┤Тќ░#

сѓ│сЃ│сЃєсЃісЂ«сЃфсѓйсЃ╝сѓ╣УеГт«џсѓњDockerРЄњсѓ│сЃ│сЃєсЃіУеГт«џТЏ┤Тќ░сѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ

ТюгжаЁсЂДсЂ»сђЂсЃЏсѓ╣сЃѕсѓёС╗ќсѓ│сЃ│сЃєсЃісЂФтй▒жЪ┐сЂїсЂДсѓІсѓѕсЂєсЂфсѓ│сЃ│сЃєсЃісѓњСй┐ућесЂЎсѓІта┤тљѕсЂФсђЂсЃфсѓйсЃ╝сѓ╣сѓњтѕХжЎљсЂЋсЂЏсѓІсЂЊсЂесѓњтЅЇТЈљсЂесЂЌсЂдсЂісѓісЂЙсЂЎсђѓ

сЃфсѓйсЃ╝сѓ╣УеГт«џсѓњТЏ┤Тќ░сЂЎсѓІсѓ│сЃ│сЃєсЃісѓњсЃЌсЃФсЃђсѓдсЃ│сЃАсЃІсЃЦсЃ╝сЂІсѓЅжЂИТіъсЂЌсЂЙсЂЎсђѓ
т»ЙУ▒Асѓ│сЃ│сЃєсЃіжЂИТіътЙїсђЂсѓфсЃЌсѓисЃДсЃ│сЃЋсѓЕсЃ╝сЃатЈісЂ│ТЏ┤Тќ░сЃюсѓ┐сЃ│сѓњжЂИТіъсЂДсЂЇсѓІсѓѕсЂєсЂФсЂфсѓісЂЙсЂЎсђѓ
сЃфсѓйсЃ╝сѓ╣тцЅТЏ┤ућесЂ«сѓфсЃЌсѓисЃДсЃ│сѓњтЁЦтіЏсЂЌсђЂТЏ┤Тќ░сЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂФсѓѕсѓіжЂЕућесЂЋсѓїсЂЙсЂЎсђѓ

ТюгТЕЪУЃйсЂДсЂ»сђЂтєЁжЃеуџёсЂФdocker updateсѓ│сЃъсЃ│сЃЅсѓњСй┐ућесЂЌсЂдсЂісѓісЂЙсЂЎсђѓ
сЂЮсЂ«сЂЪсѓЂсђЂсѓфсЃЌсѓисЃДсЃ│сЃЋсѓЕсЃ╝сЃасЂФсЂ»docker updateсѓ│сЃъсЃ│сЃЅсЂ«сЃфсѓйсЃ╝сѓ╣тѕХтЙАу│╗сЂ«сѓфсЃЌсѓисЃДсЃ│жЃесЂ«сЂ┐сѓњУеўУ╝ЅсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

  • ТюђтцДсЃАсЃбсЃфсѓњ50MbyteсЂФтѕХжЎљсЂЎсѓІта┤тљѕсЂ«сѓхсЃ│сЃЌсЃФ
    • doceker updateсѓ│сЃъсЃ│сЃЅТЎѓсЂ«СЙІ
# docker update --memory 50M --memory-swap 100M nginx_container
    • сѓфсЃЌсѓисЃДсЃ│сЃЋсѓЕсЃ╝сЃасЂФтЁЦтіЏсЂЎсѓІта┤тљѕсЂ«СЙІ
--memory 50M --memory-swap 100M

сЂЙсЂЪсђЂdocker updateсѓ│сЃъсЃ│сЃЅсЂ«УЕ│у┤░сЂФсЂцсЂёсЂдсЂ»С╗ЦСИІсЂ«сЃџсЃ╝сѓИсѓњсЂћуб║УфЇсЂЈсЂасЂЋсЂёсђѓ

Dockerсѓ│сЃ│сЃєсЃітѕЮтЏъУхитІЋ#

сѓ│сЃ│сЃєсЃісЂ«СйюТѕљсЃ╗УхитІЋсѓњDockerРЄњсѓ│сЃ│сЃєсЃітѕЮтЏъУхитІЋсѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ
сЃГсЃ╝сѓФсЃФуњ░тбЃсЂФт»ЙУ▒АсѓцсЃАсЃ╝сѓИсЂїтГўтюесЂЌсЂфсЂёта┤тљѕсђЂУЄфтІЋуџёсЂФDocker HubсЂІсѓЅтЈќтЙЌсѓњУАїсЂёсЂЙсЂЎсђѓ

сЃЌсЃЕсѓцсЃЎсЃ╝сЃѕсЃгсѓИсѓ╣сЃѕсЃфсѓхсЃ╝сЃљсЂ«сѓцсЃАсЃ╝сѓИсѓњСй┐ућесЂЎсѓІта┤тљѕсЂФсЂ»сђЂсЃђсѓдсЃ│сЃГсЃ╝сЃЅсѓ┐сЃќсЂФсЂдС║ІтЅЇсЂФсѓцсЃАсЃ╝сѓИсѓњтЈќтЙЌсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

  • сѓцсЃАсЃ╝сѓИ

СйюТѕљтЈісЂ│УхитІЋсЂЎсѓІсѓ│сЃ│сЃєсЃісЂ«сѓцсЃАсЃ╝сѓИтљЇсѓњТїЄт«џсЂЌсЂЙсЂЎсђѓ

  • RUNсѓфсЃЌсѓисЃДсЃ│

docker runсѓ│сЃъсЃ│сЃЅсЂ«сѓфсЃЌсѓисЃДсЃ│сѓњТїЄт«џсЂЌсЂЙсЂЎсђѓсЂфсЂісђЂт╝итѕХуџёсЂФсЃЄсѓ┐сЃЃсЃЂсѓфсЃЌсѓисЃДсЃ│(-d)сЂїтєЁжЃеуџёсЂФС╗ўСИјсЂЋсѓїсЂЙсЂЎсђѓ

DockerУЄфСйЊсЂ»сѓцсЃ│сѓ╣сЃѕсЃ╝сЃФ/тєЇУхитІЋтЙїсђЂтИИТЎѓУхитІЋсЂЌсЂдсЂісѓісЂЙсЂЎсђѓWEB-UIсЂДсЂ»DockerТюгСйЊсЂ«тєЇУхитІЋуГЅсѓњСИђжЃесЂ«С┐ЮтГўсѓцсЃЎсЃ│сЃѕсѓбсѓ»сѓисЃДсЃ│сЂФсЂдУАїсЂєсЂЊсЂесЂїсЂѓсѓісЂЙсЂЎсђѓ
сЂЮсЂ«сЂЪсѓЂсђЂСй┐ућесЂЎсѓІсѓ│сЃ│сЃєсЃісЂ»сЃЄсѓ┐сЃЃсЃЂ("docker run -d")тЈісЂ│тИИТЎѓтєЇУхитІЋсЃЮсЃфсѓисЃ╝("docker run --restart=always")сѓфсЃЌсѓисЃДсЃ│сѓњжЂЕућесЂЌжЂІућесЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

сѓ│сЃ│сЃєсЃісЂ«у«АуљєСИісђЂсѓ│сЃ│сЃєсЃітљЇсЂ«сѓфсЃЌсѓисЃДсЃ│ТїЄт«џ(--name <сѓ│сЃ│сЃєсЃітљЇ>)сЂФсЂцсЂёсЂдсѓѓСй┐ућесѓњТјетЦесЂёсЂЪсЂЌсЂЙсЂЎсђѓ

сЂЙсЂЪсђЂdocker runсѓ│сЃъсЃ│сЃЅсЂ«УЕ│у┤░сЂФсЂцсЂёсЂдсЂ»С╗ЦСИІсЂ«сЃџсЃ╝сѓИсѓњсЂћуб║УфЇсЂЈсЂасЂЋсЂёсђѓ

http://docs.docker.jp/engine/reference/commandline/run.html

DockerсЃГсЃ╝сѓФсЃФсѓцсЃАсЃ╝сѓИуб║УфЇ#

сЃГсЃ╝сѓФсЃФсЂФтГўтюесЂЎсѓІDockerсѓ│сЃ│сЃєсЃісѓцсЃАсЃ╝сѓИсЂ«СИђУдДсЂ«уб║УфЇсѓњDockerРЄњсѓцсЃАсЃ╝сѓИСИђУдДсѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ

сѓ│сЃ│сЃєсЃісЂ«сѓѓсЂесЂесЂфсѓІсѓцсЃАсЃ╝сѓИсЂ«СИђУдДсЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ

Сй┐ућеСИГ/ТюфСй┐ућесЂесѓѓсЂФтЅіжЎцсЃюсѓ┐сЃ│сЂїУАеуц║сЂЋсѓїсЂЙсЂЎсЂїсђЂСй┐ућеСИГсЂ«сѓцсЃАсЃ╝сѓИсЂ»тЅіжЎцсЂїУАїсЂѕсЂЙсЂЏсѓЊсђѓ
сЂЮсЂ«сЂЪсѓЂсђЂтЅіжЎцсѓњУАїсЂєта┤тљѕсЂФсЂ»сѓ│сЃ│сЃєсЃіСИђУдДсЂФсЂдС║ІтЅЇсЂФт»ЙУ▒АсѓцсЃАсЃ╝сѓИсѓњСй┐ућесЂЌсЂдсЂёсѓІсѓ│сЃ│сЃєсЃісѓњтЅіжЎцсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

DockerсѓцсЃАсЃ╝сѓИсЂ«Тцюу┤б#

Docker HubсЂФтЁгжќІсЂЋсѓїсЂдсЂёсѓІсѓ│сЃ│сЃєсЃісѓцсЃАсЃ╝сѓИсѓњDockerРЄњсѓцсЃАсЃ╝сѓИТцюу┤бсѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ

Тцюу┤бсѓГсЃ╝сЃ»сЃ╝сЃЅсЃЋсѓЕсЃ╝сЃасЂФТцюу┤бт»ЙУ▒АТќЄтГЌтѕЌсѓњтЁЦтіЏтЙїсђЂТцюу┤бсЃюсѓ┐сЃ│сѓњсЂісЂЎсЂЊсЂесЂФсѓѕсѓіТцюу┤бсЂЋсѓїсЂЙсЂЎсђѓ
сЂфсЂісђЂТюђтцДТцюу┤бС╗ХТЋ░сЂ»100С╗ХсЂЙсЂДсЂесЂфсЂБсЂдсЂісѓісЂЙсЂЎсђѓ

СИітЏ│сЂ»"nginx"сЂ«сѓцсЃАсЃ╝сѓИсѓњТцюу┤бсЂЌсЂЪСЙІсЂесЂфсѓісЂЙсЂЎсђѓ

сЃђсѓдсЃ│сЃГсЃ╝сЃЅсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂФсѓѕсѓісђЂт»ЙУ▒АсЂ«сѓцсЃАсЃ╝сѓИсѓњсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЂЎсѓІсЂЊсЂесЂїУАїсЂѕсЂЙсЂЎсђѓ
сЂфсЂісђЂт»ЙУ▒АсѓцсЃАсЃ╝сѓИсѓњсЃђсѓдсЃ│сЃГсЃ╝сЃЅСИГсЂФсЂ»жђ▓ТЇЌуіХТ│Ђсѓњуб║УфЇсЂДсЂЇсѓІсѓѕсЂєТцюу┤бсЃюсѓ┐сЃ│сЂ«жџБсЂФуіХТ│ЂУАеуц║сЃюсѓ┐сЃ│сЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓсЂЊсЂАсѓЅсЂ«сЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂджђ▓ТЇЌуіХТ│ЂсЂ«уб║УфЇсѓњУАїсЂБсЂдсЂЈсЂасЂЋсЂёсђѓ

DockerсѓцсЃАсЃ╝сѓИсЂ«сЃђсѓдсЃ│сЃГсЃ╝сЃЅ#

Docker HubсЂФтЁгжќІсЂЋсѓїсЂдсЂёсѓІсѓ│сЃ│сЃєсЃісѓцсЃАсЃ╝сѓИсѓёсЃЌсЃЕсѓцсЃЎсЃ╝сЃѕсЃгсѓИсѓ╣сЃѕсЃфсѓхсЃ╝сЃљсЂФтГўтюесЂЎсѓІсѓ│сЃ│сЃєсЃісѓцсЃАсЃ╝сѓИсЂ«сЃђсѓдсЃ│сЃГсЃ╝сЃЅсѓњDockerРЄњсЃђсѓдсЃ│сЃГсЃ╝сЃЅсѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ
сЃђсѓдсЃ│сЃГсЃ╝сЃЅсЂЎсѓІсѓ│сЃ│сЃєсЃісѓцсЃАсЃ╝сѓИсЂїсЃЌсЃЕсѓцсЃЎсЃ╝сЃѕсЃгсѓИсѓ╣сЃѕсЃфсѓхсЃ╝сЃљсЂ«та┤тљѕсђЂС║ІтЅЇсЂФт»ЙУ▒АсѓхсЃ╝сЃљсЃ╝сЂИсЂ«сЃГсѓ░сѓцсЃ│ТЃЁта▒сЂїт┐ЁУдЂсЂесЂфсѓісЂЙсЂЎсђѓсЂЊсЂАсѓЅсЂФсЂцсЂёсЂдсЂ»сЃгсѓИсѓ╣сЃѕсЃфУ┐йтіасѓ┐сЃќсЂФсЂдсЃГсѓ░сѓцсЃ│ТЃЁта▒сѓњУеГт«џсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

сЃђсѓдсЃ│сЃГсЃ╝сЃЅсЂЌсЂЪсЂёсѓцсЃАсЃ╝сѓИтљЇсѓњТїЄт«џсЂЌсЂЙсЂЎсђѓ
уЅ╣т«џсЂ«сѓ┐сѓ░сЂ«сѓцсЃАсЃ╝сѓИсѓњтЈќтЙЌсЂЎсѓІта┤тљѕсЂФсЂ»сђЂсѓцсЃАсЃ╝сѓИтљЇ[:сѓ┐сѓ░тљЇ]сѓњТїЄт«џсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

сѓцсЃАсЃ╝сѓИтљЇтЁЦтіЏтЙїсђЂсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂФсѓѕсѓіт»ЙУ▒АсЂ«сѓцсЃАсЃ╝сѓИсЂїсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЂЋсѓїсЂЙсЂЎсђѓ

сЃђсѓдсЃ│сЃГсЃ╝сЃЅсЃюсѓ┐сЃ│сѓњТі╝сЂЌсЂЪта┤тљѕсђЂуіХТ│Ђуб║УфЇсЃюсѓ┐сЃ│сЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓсЂЊсЂ«сЃюсѓ┐сЃ│сЂДжђ▓УАїуіХТ│ЂсЂїуб║УфЇсЂДсЂЇсЂЙсЂЎсЂ«сЂДсђЂсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЂЎсѓІсѓцсЃАсЃ╝сѓИсѓхсѓцсѓ║сЂїтцДсЂЇсЂёта┤тљѕуГЅсЂФсЂћтѕЕућесЂЈсЂасЂЋсЂёсђѓ

Dockerсѓ│сЃ│сЃєсЃісЂ«сЃГсѓ░уб║УфЇ#

Сй┐ућесЂЌсЂдсЂёсѓІсѓ│сЃ│сЃєсЃісЂїтЄ║тіЏсЂЌсЂдсЂёсѓІсЃГсѓ░сЂ«уб║УфЇсѓњDockerРЄњсЃГсѓ░сѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ
сЂфсЂісђЂсѓ│сЃ│сЃєсЃісѓњтЅіжЎцсЂЌсЂЪта┤тљѕсЂФсЂ»т»ЙУ▒Асѓ│сЃ│сЃєсЃісЂ«сЃГсѓ░сЂФсЂцсЂёсЂдсѓѓтЅіжЎцсЂЋсѓїсѓІуѓ║сђЂТюгжаЁсЂДсЂ«уб║УфЇсѓѓУАїсЂѕсЂфсЂЈсЂфсѓісЂЙсЂЎсђѓ

сЃГсѓ░сѓњуб║УфЇсЂЌсЂЪсЂёсѓ│сЃ│сЃєсЃісѓњсЃЌсЃФсЃђсѓдсЃ│сЃАсЃІсЃЦсЃ╝сЂІсѓЅжЂИТіъсЂЌсЂЙсЂЎсђѓ
сѓ│сЃ│сЃєсЃісѓњжЂИТіъсЂЌсЂЪта┤тљѕсђЂсЃГсѓ░УАеуц║жаўтЪЪсЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ сЂЙсЂЪсђЂт»ЙУ▒Асѓ│сЃ│сЃєсЃісЂ«ТюђухѓсЃГсѓ░сЂІсѓЅСИђт«џжЄЈУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ

Dockerсѓ│сЃ│сЃєсЃісЂ«сЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»УеГт«џ#

СйюТѕљсЂЌсЂЪсѓ│сЃ│сЃєсЃісѓњсЃЄсЃЋсѓЕсЃФсЃѕсЂ«сЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»С╗ЦтцќсѓњућесЂёсЂЪсЂёта┤тљѕсђЂсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»УЄфСйЊсЂ«СйюТѕљтЈісЂ│сѓ│сЃ│сЃєсЃісЂесЂ«ТјЦуХџсЃ╗тѕЄТќГсЂ«УеГт«џсѓњDockerРЄњсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сѓ┐сЃќсЂФсЂдУАїсЂѕсЂЙсЂЎсђѓ

сѓ┐сЃќсѓњжЂИТіъсЂЎсѓІсЂеСИітЏ│сЂ«сѓѕсЂєсЂФУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ

  • сѓ│сЃ│сЃєсЃісЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»СйюТѕљ (docker network createсѓ│сЃъсЃ│сЃЅ)

СйюТѕљсЂЌсЂЪсЂёсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»тљЇтЈісЂ│СйюТѕљсѓфсЃЌсѓисЃДсЃ│сѓњТїЄт«џсЂЌсђЂућЪТѕљсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂДсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сЂїућЪТѕљсЂЋсѓїсЂЙсЂЎсђѓ

  • сѓ│сЃ│сЃєсЃісЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»ТјЦуХџ (docker network connectсѓ│сЃъсЃ│сЃЅ)

ТјЦуХџсЂЌсЂЪсЂёсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»тљЇтЈісЂ│т»ЙУ▒Асѓ│сЃ│сЃєсЃісђЂТјЦуХџсѓфсЃЌсѓисЃДсЃ│сѓњТїЄт«џсЂЌсђЂТјЦуХџсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂДсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сЂїТјЦуХџсЂЋсѓїсЂЙсЂЎсђѓ

  • сѓ│сЃ│сЃєсЃісЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»тѕЄТќГ (docker network disconnectсѓ│сЃъсЃ│сЃЅ)

тѕЄТќГсЂЌсЂЪсЂёсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»тљЇтЈісЂ│т»ЙУ▒Асѓ│сЃ│сЃєсЃісѓњТїЄт«џсЂЌсђЂтѕЄТќГсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂДсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сЂїтѕЄТќГсЂЋсѓїсЂЙсЂЎсђѓ

  • сЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»СИђУдД

DockerсЂ«сЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сЂесЂЌсЂдтГўтюесЂЌсЂдсЂёсѓІсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сЂ«СИђУдДсѓњУАеуц║сЂЌсЂЙсЂЎсђѓ
СйюТѕљсЂЌсЂЪсЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»сѓњтЅіжЎцсЂЌсЂЪсЂёта┤тљѕсЂФсЂ»сђЂтЅіжЎцсЃюсѓ┐сЃ│сѓњТі╝сЂЎсЂЊсЂесЂФсѓѕсѓітЅіжЎцтЈ»УЃйсЂДсЂЎсђѓ
