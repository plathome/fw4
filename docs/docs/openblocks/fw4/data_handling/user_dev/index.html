сЃЄсЃљсѓцсѓ╣УеГт«џ(сЃдсЃ╝сѓХсЃ╝т«џуЙЕ)

OpenBlocksсѓисЃфсЃ╝сѓ║сЂ«ТеЎТ║ќсЃЄсЃ╝сѓ┐тЈјжЏєсѓбсЃЌсЃфсѓ▒сЃ╝сѓисЃДсЃ│(PD Handler)сѓњућесЂёсЂџсђЂсЃдсЃ╝сѓХсЃ╝сЂїуІгУЄфсЂФућеТёЈсЂЎсѓІсѓбсЃЌсЃфсѓ▒сЃ╝сѓисЃДсЃ│сЂФсѓѕсѓісЃЄсЃ╝сѓ┐сѓњсЃЈсЃ│сЃЅсЃфсЃ│сѓ░сЂЌсђЂPD RepeaterсЂежђБТљ║сЂЋсЂЏсѓІсЂЊсЂесЂїсЂДсЂЇсЂЙсЂЎсђѓ
WEB-UIсЂ«IoTсЃЄсЃ╝сѓ┐РєњжђЂтЈЌС┐АУеГт«џсѓ┐сЃќсЂ«сЃЄсЃљсѓцсѓ╣УеГт«џ№╝ѕсЃдсЃ╝сѓХсЃ╝т«џуЙЕ№╝ЅсЃАсЃІсЃЦсЃ╝сѓѕсѓісђЂсЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂ«УеГт«џсѓњУАїсЂєсЂЊсЂесЂїтЄ║ТЮЦсЂЙсЂЎсђѓ
сЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сѓњтѕЕућесЂЎсѓІсЂЪсѓЂсЂФсЂ»сђЂWEB-UIсЂ«сѓхсЃ╝сЃЊсѓ╣РєњтЪ║ТюгРєњсЃдсЃ╝сѓХсЃ╝сЃЄсЃљсѓцсѓ╣уЎ╗жї▓сѓ┐сЃќсѓѕсѓісђЂсѓхсЃ╝сЃЊсѓ╣ТЕЪУЃйсЂ«тЪ║ТюгУеГт«џсЂесЂЌсЂдсЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂїуЎ╗жї▓сЂЋсѓїсЂдсЂёсѓІт┐ЁУдЂсЂїсЂѓсѓісЂЙсЂЎсђѓсѓхсЃ╝сЃЊсѓ╣ТЕЪУЃйсЂ«тЪ║ТюгУеГт«џсЂФсЂцсЂёсЂдсЂ»сђЂсѓ╣сѓ┐сЃ╝сЃѕсѓбсЃЃсЃЌсѓгсѓцсЃЅ сЂ«сѓхсЃ╝сЃЊсѓ╣ТЕЪУЃйсѓњтЈѓуЁДсЂЌсЂдСИІсЂЋсЂёсђѓ
сЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂ«уЎ╗жї▓сЂесЂ»сђЂсЃЄсЃљсѓцсѓ╣уЋфтЈисѓњтЅ▓сѓіТї»сѓісЃАсЃбТЃЁта▒сѓњС╗ўСИјсЂЎсѓІСйюТЦГсЂДсЂЎсђѓ

уЎ╗жї▓ТИѕсЂ«сЃдсЃ╝сѓХсЃ╝т«џуЙЕсЂ«сЃЄсЃљсѓцсѓ╣сЂїтГўтюесЂЌсЂдсЂёсѓІта┤тљѕсђЂтѕЮТюЪуіХТЁІсЂДсЂ»СИітЏ│сЂ«сѓѕсЂєсЂФсЂфсЂБсЂдсЂёсЂЙсЂЎсђѓ(сЃЄсЃљсѓцсѓ╣сЂї1тђІуЎ╗жї▓сЂЋсѓїсЂдсЂёсѓІта┤тљѕсЂДсЂЎсђѓ)

сЃЄсЃљсѓцсѓ╣Т»јсЂФжђЂС┐Ат»ЙУ▒АжаЁуЏ«сЂФсЂджђЂС┐АсЂЎсѓІсѓњжЂИТіъсЂЎсѓІсЂетљёУеГт«џжаЁуЏ«сЂїУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ
жђЂС┐Ат»ЙУ▒АСИђТІгТюЅті╣сђЂжђЂС┐Ат»ЙУ▒АСИђТІгуёАті╣сЃюсѓ┐сЃ│сЂФсЂдтЁесЂдсЂ«уЎ╗жї▓ТИѕсЂ«сЃЄсЃљсѓцсѓ╣сЂ«жђЂС┐Ат»ЙУ▒АсѓњтѕХтЙАсЂДсЂЇсЂЙсЂЎсђѓ

жђЂС┐АсЂЎсѓІсѓњжЂИТіъсЂЎсѓІсЂетљёУеГт«џжаЁуЏ«сЂїСИітЏ│сЂ«сѓѕсЂєсЂФУАеуц║сЂЋсѓїсЂЙсЂЎсђѓ
