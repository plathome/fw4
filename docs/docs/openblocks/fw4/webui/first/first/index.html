сЂ»сЂўсѓЂсЂФ#

ТюгТЏИсЂ»сђЂOpenBlocksсѓисЃфсЃ╝сѓ║сѓњ WEB сЃдсЃ╝сѓХсЃ╝сѓцсЃ│сѓ┐сЃ╝сЃЋсѓДсЃ╝сѓ╣(С╗ЦСИІсђЂWEB-UI)Сй┐сЂБсЂдсЂ«сѓ╣сѓ┐сЃ╝сЃѕсѓбсЃЃсЃЌсЂФжќбсЂЎсѓІТЅІжаєсѓњУДБУфгсЂЌсЂдсЂёсЂЙсЂЎсђѓ
сЂЙсЂЪсђЂТюгУеГт«џсЂФсЂ»WEBсЃќсЃЕсѓдсѓХсЂїСй┐ућетЈ»УЃйсЂфсѓ»сЃЕсѓцсѓбсЃ│сЃѕУБЁуй«(PCсѓёсѓ╣сЃъсЃ╝сЃѕсЃЋсѓЕсЃ│сђЂсѓ┐сЃќсЃгсЃЃсЃѕуГЅ)сЂїт┐ЁУдЂсЂФсЂфсѓісЂЙсЂЎсђѓ
сЂфсЂісђЂТюгТЏИсЂДсЂ»сЃЉсѓйсѓ│сЃ│сЂ«WEBућ╗жЮбсѓњућесЂёсЂдУДБУфгсѓњУАїсЂёсЂЙсЂЎсђѓ

SIMсЂ«Т║ќтѓЎ#

сЃЇсЃЃсЃѕсЃ»сЃ╝сѓ»СИіТхЂсЂИТјЦуХџсЂЎсѓІТЎѓсђЂСИіТхЂсЂїLTEсЂфсЂЕсЂ«сЃбсЃљсѓцсЃФтЏъуиџсЂ«та┤тљѕсЂ»С║ѕсѓЂSIMсѓњсѓ╗сЃЃсЃѕсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
SIMсЂ«сѓ╗сЃЃсЃѕТќ╣Т│ЋсЂ»УБйтЊЂТи╗С╗ўсЂ«ТЏИжЮбсђїсЂћСй┐ућесЂФсЂѓсЂЪсЂБсЂдсђЇсЂЙсЂЪсЂ»ТюгсЃъсЃІсЃЦсѓбсЃФсЂ«сђїсЃЈсЃ╝сЃЅсѓдсѓДсѓбсђЇуиесЂФсЂѓсѓІТЕЪуе«тѕЦТ»јсЂ«сђїсЂћСй┐ућесЂФсЂѓсЂЪсЂБсЂдсђЇсЂ«сЃфсЃ│сѓ»сЂїсЂѓсѓІсЂ«сЂДсЂЮсѓїсѓњсЂћУдДсЂЈсЂасЂЋсЂёсђѓ
сђїсЂћСй┐ућесЂФсЂѓсЂЪсЂБсЂдсђЇсЂ«SIMТї┐тЁЦТќ╣Т│ЋсЂФтЙЊсЂёсђЂУБЈУАесѓњжќЊжЂЋсЂѕсЂфсЂёсѓѕсЂєсѓ╣сЃГсЃЃсЃѕсЂИТї┐тЁЦсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ

SIMсЂ«сѓхсѓцсѓ║

OpenBlocksсЂїТјАућесЂЌсЂдсЂёсѓІSIMсЂФсЂ»тцДсЂЇсЂЋсЂї2уе«жАъсЂѓсѓісЂЙсЂЎсђѓ
ТЕЪуе«Т»јсЂФтЪ║ТЮ┐сЃљсЃ╝сѓИсЃДсЃ│сЂїтЈцсЂёсѓѓсЂ«сЂФсЂ»mini-SIM(ТЌЦТюгсЂДсЂ»сЂЊсѓїсѓњТеЎТ║ќSIMсЂетЉ╝сѓЊсЂДсЂёсѓІсђѓ)сЂ«сѓ╣сЃГсЃЃсЃѕсЂїТјАућесЂЋсѓїсђЂТќ░сЂЌсЂётЪ║ТЮ┐сЃљсЃ╝сѓИсЃДсЃ│сЂФсЂ»nano-SIM№╝ѕТюђУ┐ЉсЂ«сѓ╣сЃъсЃ╝сЃѕсЃЋсѓЕсЃ│сЂДућесЂёсѓЅсѓїсЂдсЂёсѓІсѓхсѓцсѓ║№╝ЅсЂ«сѓ╣сЃГсЃЃсЃѕсЂїТјАућесЂЋсѓїсЂдсЂёсЂЙсЂЎсђѓ
ТюђУ┐ЉсЂ«сЃбсЃљсѓцсЃФтЏъуиџТЦГУђЁсЂ»сђЂсѓ»сЃгсѓИсЃЃсЃѕсѓФсЃ╝сЃЅсѓхсѓцсѓ║сЂ«сЃЌсЃЕсѓ╣сЃЂсЃЃсѓ»ТЮ┐сЂІсѓЅSIMсѓњтЦйсЂЇсЂфсѓхсѓцсѓ║сЂДтѕЄсѓіТгасЂёсЂдСй┐сЂѕсѓІсЃъсЃФсЃЂсѓФсЃЃсЃѕсѓ┐сѓцсЃЌсЂДСЙЏухдсЂЌсЂдсЂёсЂЙсЂЎсђѓ
сЃъсЃФсЃЂсѓФсЃЃсЃѕсѓ┐сѓцсЃЌсЂ«та┤тљѕсђЂmini-SIMсѓхсѓцсѓ║сЂ«та┤тљѕсЂ»СИђуЋфтцДсЂЇсЂфтѕЄсѓіТгасЂЇсЂДSIMсѓњТіюсЂЇтЈќсѓісЂЙсЂЎсђѓ
