WEB-UIタブ別インデックス

ダッシュボード#

リソースの使用状況とプロセス起動停止操作

サービス#

OpenBlocksに追加されたサービスガイド

システム#

OpenBlocksの基本システムの操作

基本#
  • 時刻設定(タイムサーバー設定)とGPS座標(設置場所)の登録
詳細#
  • 機能設定(WEB-UIタブのマスク)
  • 機能割り当て(OpenBlocks本体のFUNCスイッチ)
  • プロセス状況表示(ユーザー定義)
  • ストレージ管理(メール通知)
  • ストレージクリーンナップ(有効化)
  • 電源監視(停電監視と内蔵バッテリー)
  • 本体自動再起動設定(有効化)
パスワード#
  • rootパスワードの設定
フィルター#
  • サービス毎のポートフィルターの設定
SSH関連#
  • SSHの設定詳細、鍵付きSSHの設定
マイページ#
  • WEB-UIの管理者パスワードの設定
WEBユーザー#
  • WEB-UI閲覧ユーザーのアカウント管理(管理者よりも表示制限が多)
ファイル管理#
  • OpenBlocks内のファイルとディレクトリ管理およびクライアントパソコンとのファイル転送
ライセンス#
  • オープンソースライセンスの表示
SYSLOG転送#
  • OpenBlocksのSyslogをSyslogサーバーに転送します。
FSTAB編集#
  • fstabを編集
    ※ラベル付きのUSBストレージは自動マウントされます。
S/N#
  • OpenBlocksのシリアルナンバーを表示

ネットワーク#

基本#
  • ホスト名、ネットワークドメイン登録、無線LAN、イーサネット、モバイル回線の設定
HTTPプロキシ#
  • 上流ネットワークへのプロキシ設定
ホスト追加#
  • 接続先ホストのIPアドレスを登録
WLAN証明書#
  • 無線LANの証明書の登録
ダイナミックDNS#
  • ダイナミックDNSの登録
ルーティング#
  • スタティックルーティングの設定
通信確認#
  • pingなどでの通信先の確認
状態#
  • ネットワークステータスの一覧

メンテナンス#

設定#
  • OpenBlocksのconfig情報をクライアントパソコンにエクスポート、またはクライアントパソコンからインポート
システムの更新#
  • OpenBlocksのシステムソフトウェアの更新
    ※リポジトリサーバーの登録でユーザーアプリも更新可能
停止・再起動#
  • OpenBlocksのOSの停止・再起動
サポート#
  • サポート用全ログをクライアントパソコンで取得
機能拡張#
  • OpenBlocksへサービスや拡張サービスを追加インストール
Pubキー追加#
  • Node.jsなどのパブリックキーを登録

拡張#

スクリプト編集#
  • 起動スクリプトの追加やユーザー定義スクリプト、OpenBlocks本体のFUNCキーのスクリプトを編集
コマンド実行#
  • Linuxコマンドのダイレクト実行
SMSコマンド実行#
  • SMSでのメッセージ受信によるユーザー定義スクリプトの実行
SSHトンネル#
  • 他のサーバーとSSHトンネルを張る
監視#
  • ログのアラート設定やプロセスの監視設定
URIプロキシ#
  • ローカルネットワークの指定のipアドレス機器のhttp/httpsポートをAirManageに転送
稼働情報#
  • プロセスなどの稼働状況をjsonファイルに記録します。(連携アプリ別途必要)
Samba#
  • Windowsパソコンとファイル共有
FTP#
  • FTPサーバーを設定
FTPダウンロード#
  • FTPクライアントとして一定間隔でファイルをFTPサーバーからダウンロードします。
FTPアップロード#
  • FTPクライアントとして一定間隔でOpenBlocks内に置かれたファイルをFTPサーバーへアップロードします。

技術情報#

  • OpenBlocksのAirManage登録や、ユーザーマニュアル、FWアップデートなどの情報ページへのリンクページです。
