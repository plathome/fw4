OpenBlocks HX1 Debian搭載モデル向け

製品型番#

型番備考
OBSHX1/DnAAirManage Agent搭載版のOpenBlocks HX1 Debian搭載モデル
info

型番のnはDebianのメジャーバージョンを数字で表します。
Debian 11のAirManage Agent搭載版のOpenBlocks HX1はOBSHX1/D11Aとなります。

出荷時情報設定#

シリアルコンソール管理者ユーザ#

項目値備考
ユーザ名root
パスワード0BS|-|Xl
  • 1文字目は数字の「0」
  • 4文字目・6文字目は記号の「|」(パイプ)
  • 5文字目は記号の「-」(ハイフンマイナス)
  • 8文字目はアルファベットの「l」(小文字のエル)

AirManage用一般ユーザ#

項目値備考
ユーザ名obsuser
パスワード0penB1ox1文字目は数字の「0」、5文字目は数字の「1」
info

パスワードの変更を推奨いたします。
